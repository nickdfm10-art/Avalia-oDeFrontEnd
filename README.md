# InstaOptimize

## Nome do Projeto

InstaOptimize – Agência de Otimização de Conteúdo Digital

---

## Objetivo

O projeto InstaOptimize foi desenvolvido com o objetivo de apresentar os serviços de uma agência especializada na otimização de conteúdo para Instagram. O site busca demonstrar como a adequação de imagens, perfis e postagens pode melhorar a presença digital de empresas, criadores de conteúdo e pequenos empreendedores.

Além da apresentação institucional da agência, o projeto aplica os conceitos fundamentais de desenvolvimento web utilizando HTML, CSS e JavaScript puro.

---

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---

## Estrutura das Páginas/Seções

### Início

Apresentação da empresa e de sua proposta de valor, destacando os serviços de otimização para Instagram.

### Nossas Soluções

Apresentação dos principais serviços oferecidos pela agência:

* Imagem de Perfil Profissional
* Otimização de Postagens
* Padronização Visual do Feed
* Conteúdo para Dispositivos Móveis

### Clientes

Seção destinada à apresentação de exemplos de clientes atendidos e dos serviços realizados, utilizando tabela com imagens ilustrativas dos resultados.

### Contato

Formulário para que potenciais clientes possam enviar suas informações e solicitar atendimento.

---

## Funcionalidades Implementadas

### Estruturação Semântica

Utilização das principais tags semânticas do HTML:

* `<header>`
* `<nav>`
* `<main>`
* `<section>`
* `<form>`
* `<table>`
* `<fieldset>`

### Estilização com CSS

* Personalização de cores
* Formatação de textos
* Estilização de tabelas
* Configuração de bordas
* Organização de espaçamentos
* Ajuste e dimensionamento de imagens

### Manipulação do DOM com JavaScript

* Seleção de elementos utilizando `getElementById()`
* Alteração dinâmica de conteúdo utilizando `innerHTML`
* Alteração dinâmica de estilos através do JavaScript

### Tratamento de Eventos

* Captura do evento `submit` do formulário
* Cancelamento do comportamento padrão com `preventDefault()`

### Validação de Formulário

Verificação dos campos:

* Nome
* E-mail
* Mensagem

Caso algum campo esteja vazio, o sistema exibe uma mensagem solicitando o preenchimento correto.

### Simulação de Envio

Após a validação dos dados:

* Exibe mensagem de sucesso ao usuário
* Limpa automaticamente os campos do formulário através do método `reset()`
